<h4>Задание проекта:</h4></br>
Разработать тест-дизайн и на его основе разработать позитивные API-тесты для тестирования функции фильтрации отбора животных в веб-приложении «Дом питомца». На выходе должен быть получен файл с вариантами проверок ключей species_name, breed_code, gender_code и age. Вы можете использовать разные длины строк и символы. Только помните, что в этом модуле мы разбираем позитивные тесты! Не нужно писать текстовые символы в числовые поля.</br>

Использовать приложение Postman. Тесты собрать в одну коллекцию. Ожидаемый результат — возврат API-запросом кода состояния 200. Цель — продемонстрировать навыки работы с приложением Postman.
