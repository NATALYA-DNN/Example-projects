Стажировка от компании PRO.MOTION


Шаг 1. Протестируйте поиск

Результаты поиска должны быть релевантны запросу; товары должны быть во всех магазинах. Если в каком-то маркетплейсе не оказалось товаров, то следует проверить его выдачу прямым запросом на сайт маркетплейса.
Проверьте отображение товаров для выбранного города. Для этого запросите результаты в определенном магазине. Товары сравниваются, так как парсеры параллельно выполняют запрос пользователя во всех магазинах с выбранным городом.
Если в поиске выбран, например, город Омск, то при переходе на страницу товара в магазине товар должен быть доступен для продажи в выбранном городе.
Шаг 2. Протестируйте фильтрацию цен, рейтинга, популярности, городов

Дефолтная сортировка — по популярности. Сверху должны идти товары, у которых больше всего отзывов.
Сортировка по рейтингу: сверху должны товары, у которых самый высокий рейтинг.
Сортировка по цене: изначально товары с меньшей ценой должны быть сверху, а при повторном клике на кнопку фильтра сверху должны быть, наоборот, товары с наивысшей ценой.
Фильтрация по ценовому диапазону: должны отображаться только товары, которые вписываются в указанный пользователем диапазон.
Шаг 3. Проверьте SEO (canonical link, быстрое открытие страницы, заголовки и пр.)

Каждая страница должна содержать canonical link.
Скорость ответа от всех маркетплейсов не должна превышать 20 секунд.
Все картинки должны содержать тег alt.
Шаг 4. Проверьте UI

Фото товара должно корректно отображаться и соответствовать самому товару.
При передвижении мыши по слайдеру фотографии должны меняться.
Тексты должны соответствовать и товару, и фильтру, а также не налезать друг на друга.
Шаг 5. Проверьте сужение поиска по товару

При наведении на товар из запроса должна появляться кнопка «Найти похожий».
При клике на кнопку «Найти похожий» должен происходить запрос с названием выбранного товара.
Если название товара слишком длинное, то оно должно обрезаться до 10 слов.
Шаг 6. Проверьте отзывы

При наведении на товар должна всплывать кнопка «Отзывы».
При клике на кнопку «Отзывы» должно открываться новое окно с отзывами на товар.
Фильтрация должна работать корректно: например, при клике на фильтр «С фото отображаются отзывы» должны показываться варианты только с фото.
При клике на кнопку «Показать еще» подгружаются новые отзывы.
Шаг 7. Проверьте главную страницу сайта

На главной странице должны содержаться подборки с самыми популярными поисковыми запросами.
У товара в подборке должна показываться минимальная цена (не учитывая AliExpress, потому что там попадаются копеечные товары).
Фото подборки должно выставляться от самого популярного товара (дефолтный поиск — по популярности; отображается самый продаваемый и обсуждаемый товар).
